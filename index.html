A classe .container e suas propriedades

A classe .container possui algumas propridades em relação a disposição de elementos na tela;

Tipo função
.container cria um contêiner fixo e responsivo. Isso significa que a largura do contêiner será fixa para determinados pontos de interrupção (breakpoints), e o conteúdo será centralizado dentro desse contêiner. O comportamente é similar as margens de um livro ou caderno.
.container-fluid cria um contêiner fluido, que se estende por toda a largura da janela do navegador. O conteúdo dentro deste contêiner se estenderá para ocupar toda a largura disponível.
.container-"breakpoint" A classe container-breakpoint (por exemplo, .container-sm , container-md, container-lg, container-xl) define um contêiner fixo para um ponto de interrupção específico. Isso permite controlar a largura do contêiner em diferentes tamanhos de tela.

Estilização de Formulários

O Bootstrap fornece classes e estilos que facilitam a estilização de formulários, tornando-os mais atraentes e responsivos. Abaixo estão algumas classes úteis para estilizar elementos de formulário:

  • .form-control: Aplica estilos básicos a campos de entrada.
  • .form-label: Aplica estilos básicos aos labels do formulário.
  • .btn: Estiliza botões.
  • .form-check: Estiliza caixas de seleção e botões de rádio.
  • .form-group: Agrupa elementos de formulário e fornece margens.

Estilização de Tabelas

O Bootstrap oferece classes que permitem estilizar tabelas de forma rápida e fácil. Algumas classes úteis para estilizar tabelas são:

  • .table: Aplica estilos básicos a uma tabela.
  • .table-striped: Adiciona listras alternadas às linhas da tabela.
  • .table-hover: Adiciona um efeito de destaque nas linhas quando o mouse passa sobre elas.
  • .table-bordered: Adiciona bordas às células da tabela.

Elementos de Navegação (nav) do Bootstrap

O Bootstrap oferece classes e componentes para facilitar a criação de elementos de navegação responsivos. Abaixo estão alguns dos principais elementos utilizados:

  • .navbar: Define uma barra de navegação.
  • .nav: Define uma lista de navegação dentro da barra de navegação.
  • .nav-item: Define um item da lista de navegação.
  • .fixed-top: Fixa a barra de tarefas no topo da página enquanto se realiza o scroll da página.
  • .stick-top: Faz com que o elemento ocupe 100% lateralmente da página.
